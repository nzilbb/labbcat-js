Class: GraphStore

GraphStore(baseUrl, username, password)

Read/write interaction with LaBB-CAT corpora, based on the nzilbb.ag.IGraphStore. interface

Constructor

new GraphStore(baseUrl, username, password)

Create a store client
Parameters:
Name Type Description
baseUrl string The LaBB-CAT base URL (i.e. the address of the 'home' link)
username string The LaBB-CAT user name.
password string The LaBB-CAT password.
Author:
  • Robert Fromont robert@fromont.net.nz
Source:
  • nzilbb.labbcat.js, line 416
Example
// create annotation store client
const store = new GraphStore("https://labbcat.canterbury.ac.nz", "demo", "demo");
// get a corpus
store.getCorpusIds((corpora, errors, messages, call)=>{ 
    console.log("transcripts in: " + corpora[0]); 
    store.getGraphIdsInCorpus(corpora[0], (ids, errors, messages, call, id)=>{ 
        console.log("Deleting all transcripts in " + id));
        for (i in ids) {
          store.deleteGraph(ids[i], (ids, errors, messages, call, id)=>{ 
              console.log("deleted " + id);
            });
        }
      });
  });
